## 使用方式 How it works

### 第一步：設置項目目錄 Step 1: Set your project folder

初次使用時，請先點擊左上角設置項目目錄。  
這將作為之後所有項目的存儲與讀取位置。

On first launch, set a project folder from the top-left corner.  
This will be the location used to store and load all projects.

### 第二步：新建或打開項目 Step 2: Create or open a project

點擊「新建」可創建新項目，  
點擊已有項目可直接開始編輯。

Click **New** to create a project,  
or click an existing project to start editing.

### 第三步：在文本視圖中整理想法 Step 3: Organize ideas in text view

使用 **Tab** 縮進表達層級關係。  
右側文本框支持從左側導入內容，或向左側導出內容，方便批量編輯。

Use **Tab** indentation to represent hierarchy.  
The text box on the right supports importing from and exporting to the left side for batch editing.

### 第四步：在月視圖中安排日期 Step 4: Schedule dates in month view

你可以用 `mmdd` 格式輸入起止日期。  
例如 10 月 1 日可輸入為 `1001`。

You can enter start and end dates in `mmdd` format.  
For example, October 1 is written as `1001`.

你也可以直接在甘特圖中操作：

- 雙擊創建時間段  
- 拖動以移動位置  
- 雙擊調整長度以修改排期  

You can also work directly in the Gantt chart:

- Double-click to create a time range  
- Drag to move it  
- Double-click to resize the schedule  

### 第五步：在週視圖中安排時間段 Step 5: Schedule time slots in week view

將左側事件拖入右側時間表即可完成安排。  
拖回左側即可取消安排。

Drag events from the left into the timetable on the right to schedule them.  
Drag them back to the left to remove the arrangement.
